# Groupie Tracker

A user-friendly website using a given API to display and interact with data about bands, including their history, members, concert locations, and dates, with data visualizations and a client-server interaction feature

# Author 

* [@smslash](https://github.com/smslash)

## Run 

```bash
go run ./cmd
```

Then open link [http://127.0.0.1:4000](http://127.0.0.1:4000) in your browser.

## Objectives

Groupie Trackers is a user-friendly website that leverages an API to offer a comprehensive look into band and artist data. It features a versatile search bar and filtering system, enabling users to easily find and sort artists. Additionally, its geolocation feature maps concert locations, enhancing user experience. Adhering to Schneiderman's 8 Golden Rules of Interface Design, this well-crafted platform seamlessly bridges data manipulation and visualization, providing an intuitive and engaging space for music enthusiasts.
